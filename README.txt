This is the client program's directory.

This package contains a client program that implements the TFUP and TFTP protocols. 
It is able to keep files in a target server's directory  and the local working directory, in sync.

Compile the client by running 'make' in directory 'a2source'.

The client program created will be "fsynccli". 
Run it to see usage help.

A server is also provided.

Compile the server by running 'make' in directory 'modela2Server20Oct'.
The server program created will be called "fsyncserv".
Run it to see usage help.


run 'make clean' , in either directories, for a quick clean up. see 'makefile' for building details.
